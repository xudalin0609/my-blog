# 啤酒

| 有一阵很喜欢喝啤酒，数量不多，但种类不少。还学着别人做些记录，现在来看，不过是东施效颦。

## 道酿咖啡赛松

酒精度4.5 虽然叫做咖啡赛松，但却没有丝毫咖啡影子，反而有一些咸咸的味道。闻起来香气不足，毕竟是赛松而不是ipa这类的啤酒。值得一喝，口味较淡，适合作为第一杯啤酒

### 打嗝海狸世涛

酒精度4.5 淡巧克力味道的世涛 口感比较清爽，酒精度也低。闻起来味道香气清淡。入口也是缺了些特点，巧克力味道也不浓厚。回味有一点的甜，容易入口。

## 阿卡德米啤酒坊（ipa）

酒体橙红色，酒花粗糙，酒体单薄。闻起来没有ipa的典型香气，猜测为单料，沙口感较轻，苦度低，回味有甜味，有麦香（感觉窜台到小麦了）。反上来的酒气倒是有些许果香，不推荐。

直接对比酿酒狗朋克，酒体色彩相似，但更为清亮，酒体更加厚重，酒沫细腻，热带水果香气浓，沙口感极轻，苦度中，回口干净。

## 大道APA

酒花薄，但细腻。香气极淡（香蕉香气）。苦度中等，入口苦，回口天。啤酒花的作用没有丝毫体现。毫无意思的一款酒。
