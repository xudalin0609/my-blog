# 写于31岁生日

不知不觉间又是一年，去年的30岁还在上海，和朋友们吃吃喝喝之后，坐在回家的车上，看着路灯向着身后闪烁而过。而今却已经辞职近一年的时间，有些感慨，又有些不知所措。

## 日子

在家的日子自由却容易陷入混乱，偶然的一次熬夜，可能就会导致晚期，然后再晚睡，再晚起。又或者一些心理上莫名的疲惫感让自己无心于任何事情，于是决定放个假，一天，两天，三天，时间就迅速的流过了。

好在我不是个很容易沉溺于某种惯性的性子，摆烂久了就会像起来折腾一番，也因此让日子有了起起伏伏。喝茶，看书，做饭，学习，工作，爱好，运动。哪一件我都没有在这一年完完整整的坚持下来，但又不曾扔下其中的任何一件，大概是三天打鱼两天晒网，然后又跑去山上砍了三天的柴火。

于高兴时自得，寻思着“我一个人也能把生活过的如此有滋味，属实不易，厉害厉害”。于沮丧时忧心，不知未来置于何处，不知自己是否在蹉跎时光。而其中最为高兴时大概会每天自诩没有白活，满心的平和与怡然，只盼望能长久这般。至于最为沮丧时，不谈也罢。

我曾经以为是工作后的每日碌碌让自己麻木，于是加速了时光的流逝，以至于失去了关于时间的许多记忆。可现在的日子也没有放缓分毫，甚至有变快了的趋势，不只是日子过得太轻松以至于无感，还是年龄增长带来的必然的麻木。

二十岁之前的日子好长好长，长到我以为会永远留在这段时光。而三十岁也是遥远的，远到我以为可以在这段时间里完成我所有的幻想。可现在，我有些不知所措了，日子很好，可过的太快，我好像抓不住它，也留不下什么，这令我有些慌张。当然有时也会踌躇满志，寻思着，我还有方向，我还在前行，我还在生活，这很好。

所以日子就像是一个起伏的，螺旋的阶梯，而我在上面走走停停，只有走过了才知道自己走了很远很远，不曾抓住什么，但日子终归是过去了，终归是不会停下。

## 关于死亡的感受

昨天半睡半醒间没由来的生出一种悲伤，悲伤于亲人的老去甚至死亡，而死亡后便再也见不到他们了。这是一种没经历过就无法想象的痛苦，又是想起来就会如同被掏空般令人恐惧的痛苦。如同一场噩梦，梦里我犯了无法饶恕的错误，下半生皆毁于此，醒来后庆幸又后怕。

从衰老到死亡，一些不曾准备过的事情在逐渐向我逼近，然后在某一刻忽然被其捕获，纵然曾远远的看它许多次，相遇的一刻仍如初见，譬如三十岁，譬如死亡。

以前读一些心理学书籍时，曾怀疑过，童年，性，死亡，这些一般被用作心理学分析底层的东西真的那么重要么？以死亡为例，我们真的会有意无意的将死亡摆在一个很重要的位置上么？然后围绕死亡再展开一系列的行为？而如今，当我开始面对的时候，我有了些答案。

“死亡”构建了“生”的存在，而生的意义是我们最为直接的，无法逃避的问题。当面对必然的死亡时，你会如何认知生呢？死亡赋予了生怎样的意义？无意义？又或者是某些可以对抗死亡的意义？每个人在提供着答案，而这些答案便是我们行为背后的东西。

从思考死亡到理解死亡到底有多远的距离？我不知道，如果知道了会是一件令人欣喜还是难过的事情呢？我也不知道。我什么都不知道，但还是要活下去，面朝着死亡活下去。

## 工作和未来

我的物质需求并不高，一个月有个两三千块就已经够花了，所以我觉得这一年能有个五六万的收入就很好了。可眼看着目标不远了，却又不知足起来，开始担忧收入能否持久和稳定，开始厌烦工作时候的琐碎事情。比如这两天，就是因为一些事情打碎了我休息的时间，令我生出了不少厌烦的情绪，以至于干脆就不去做了。这当然是很不好的，问题不可避免，面对才能解决问题。

可这般琐碎又惹人烦恼的事情又是我辞职的原因，如果现在仍陷于其中，那我又何必辞职？思来想去，答案大概是“决定权”和“平衡”。我可以决定休息，也可以平衡工作和生活的关系，我无法逃离工作，却可以给自己一个喘息的空间。

而自由也是把双刃剑，从公司逃脱了，我变得有无限的可能，众多的可能中当然也包括颗粒无收，这无疑是令人有些恐惧的，还有关于未来的不确定性，种种因素叠加在一起，惹人烦恼。

我也可以侃侃而谈，为自己画饼，说着未来的光明和可能性，可纵然是舌绽莲花，也无法骗得自己心安。

说到底，不安的始终是我自己，惶惶之中，这也不安，那也不安，留在公司和离开公司又有什么区别么？

好在我此刻是不后悔的，我做了决定，也能为决定负责，也许前方不甚光明，但终究有路。

## 我？

我是一个自卑的人，我可以从统计学上说出自己大概位于区间的位置，但感受上却很难说出自己的优点。可我又带着些拧巴的骄傲，这份骄傲生于自卑，渴望别人的认同，骄傲的毫无底气。

这份自卑带来最大的问题大概是缺爱，我时常会拧巴的伤害自己，伤害亲密的人，然后试图通过伤害证明我能获得无条件的爱，伤害别人伤害自己。

又或者自卑的讨好身边的人，生怕留下些许厌恶。又频频将自己无端的情绪加于亲密的人。这都是很不好的。

我可能又是个孤单的人吧，不知为何始终带着些疏离感，和亲人和朋友都是这样。就像别人评价我的，始终不够了解我。是啊，我似乎并不愿意坦诚自己的悲伤和痛苦，却在悲伤和痛苦时候伤害了别人。

如果要让我给自己一个正面的评价，我更期望的词语是改变，不断地审视自己，不断地修正，不断地和解。这些都是我觉得很好的，让我知道我还在切实地活着。

关于这些生命里的种种，我应该还有许多东西能写，就留给以后吧。
