# 2025-07-21

最近的日子在持续摆烂。似乎有事情有很多事情想做，但又时常犯懒，懒得开始，开始了又静不下心，急着要一个结果，却往往被过程的枯燥劝退。虽说时间不算紧张，但想到被浪费掉的时间还是会感到些许的焦虑，有时甚至会想如果生了什么病能让我正大光明的摆烂就好了。

虽然时常对许多的事物充满好奇，但往往稍微操作几下之后就会搁置一旁，究其原因大概是急于求成。开始前想的是完成之后别人的夸赞，完成之后可以获取到的种种。可急着求一个结果，多半会半途而废。枯燥的重复和无进展的技艺都成为了坚持下去的阻力，而幻想提供的动力也仅能维持最初的那段日子。

所以我的好奇并非真的好奇，究其本质更像是虚荣和幻想凝聚出的泡泡，看起来斑斓又晶莹，实际一戳就碎。

这实在是一件可悲的事情，偏爱夸夸其谈实则空无一物。不能给自己一个骄傲的理由，也赚不到一些安身立命的本钱。

说起来，我好像也是知道如何爱上一件事情的，凭着热情进去，凭着一些坚持或者其他的什么深入，然后发现其真实的魅力，进而爱上它。可我总是停留在继续深入路上，理由也是千奇百怪，比如最近，本该每天打开电脑，学这学那，却托词客厅太热，躺在卧室吹空调玩手机。

这般的日子放在高考后的暑假应该是令人愉悦的，可到了今日，无论我嘴上说着多么的无所谓，可始终会有些关于未来的紧张感。这么说起来，倒是有些像期末前一边摆烂，一边焦虑的心态。

我希望自己是对种种事物充满热爱的，发自内心的，无关外界的热爱。偏偏心思繁杂，能量匮乏又非天资聪慧，这就让我和我的期待有了矛盾，时常陷入无意义的焦虑之中。

写到这里忽然不知道说点什么了，因为发现剖析自己实在是无聊的事情。我并不特殊，经历的事情和心态也不过世间种种的不断重复，乏味至极。又想到这些东西会被他人看到，就再加上一层负担，担心别人觉得晦涩，抽象，无聊。

表达就此陷入了个瓶颈，厌倦了自己空洞的表达，却不知道如何言之有物。也罢，就此打住，重新学习说话，重新练习表达，希望有一天我会再次觉得自己写的东西有点意思。
